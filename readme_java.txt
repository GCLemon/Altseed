■　概要
Altseedをダウンロードしていただきありがとうございます。
本パッケージには、Altseedの実行に必要なファイル及びサンプル、ヘルプ等が保存されています。

Javaはコンパイル環境が複数あり、これといった構築済みのプロジェクトがないため、サンプルはソースコードのみで提供しています。
ただ、Sampleディレクトリのbinにコンパイル済みの実行ファイルがあります。
これを実行することで大体、Altseedでできることはわかるでしょう。
その実行ファイルのソースコードはSampleディレクトリののsample_javaにあります。

プロジェクトを作成する場合は、Documentに記述されている導入手順を元にRuntimeに保存されているファイルを配置しましょう。

■　ディレクトリ構成
・Document
Altseedの導入方法や使用方法が記述されたマニュアルです。
Runtimeにあるファイルを用いてゼロからプロジェクトを構成する方法も記載されています。

・Runtime
Altseedの実行に必要なファイルです。

・Sample
Altseedを使用した基本的な機能のサンプルです。

・Tool
フォントファイルやファイルパッケージを生成するツールが保存されています。
